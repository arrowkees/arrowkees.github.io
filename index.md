# Arrowkees - アローキーズ

矢印キー付き日本語キーボード

## Features / 特徴

- **Arrow Keys** - Navigate text with arrow keys
- **Flick Input** - Japanese flick input support
- **QWERTY** - Standard English keyboard
- **Shortcuts** - ⌘+C, ⌘+V, ⌘+Z support
- **130,000+ words** - Large Japanese dictionary
